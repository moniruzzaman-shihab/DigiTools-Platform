# 🚀 DigiTools Platform

## 📌 Project Overview

DigiTools Platform is a modern web application where users can explore and purchase premium digital tools. It provides a clean UI, smooth user experience, and a simple cart system to manage selected tools.

---

## 🧠 Description

This project is designed to help users discover powerful digital tools in one place. Users can browse products, view details, add them to cart, and simulate a checkout experience. The UI is fully responsive and built with modern frontend technologies.

---

## 🛠️ Technologies Used

* ⚛️ React (Vite)
* ⚛️ JavaScript (ES6+)
* 🎨 Tailwind CSS + DaisyUI
* 🔥 Lucide React (Icons)
* 🍞 React Toastify (Notifications)

---

## ✨ Features

* 🛒 Add to Cart system with live count update
* 🎯 Dynamic product rendering from JSON data
* 📱 Fully responsive design (mobile, tablet, desktop)
